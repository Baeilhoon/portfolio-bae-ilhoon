## 💻 기술 스택

### 언어 & 개발
- **C/C++** (저수준 프로그래밍, 멀티스레드, 성능 최적화)
- **Arduino C** (MCU 펌웨어, 센서 제어, 통신)
- **Python** (데이터 처리, 시스템 테스트, 프로토타이핑)

### 임베디드 플랫폼
- **Embedded Linux**: Ubuntu 22.04 (시스템 프로그래밍)
- **MCU**: ESP32, Arduino (IoT, 센서 시스템)
- **SBC**: Raspberry Pi (Embedded AI, 데이터 수집)

### 통신 & 네트워크
- **IoT**: MQTT (Publish/Subscribe 기반 통신)
- **미들웨어**: ROS2 DDS (분산 시스템)
- **Serial**: UART, I2C, SPI (하드웨어 통신)

### 개발 도구
- **Build**: CMake, Colcon, Makefile
- **IDE**: VSCode, Arduino IDE, STM32 CubeIDE
- **Debug**: GDB, Serial Monitor, 로직 분석기
- **Version Control**: Git (협업, 브랜치 전략)
