- **Embedded Linux**: Ubuntu 22.04 기반 실시간 애플리케이션 개발
- **ROS2**: DDS 통신, 멀티노드 시스템, 데이터 동기화 최적화
- **영상 처리**: OpenCV, 카메라 제어, 실시간 데이터 파이프라인
- **문제 해결**: 성능 병목 분석, 프로파일링, 최적화 경험
