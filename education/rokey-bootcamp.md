# 🎓 두산 로보틱스 ROKEY 부트캠프

## 📋 프로그램 개요

**프로그램명**: 두산 로보틱스 ROKEY 부트캠프 5기  
**기간**: 2025년 7월 ~ 2026년 1월 (6개월)  
**기관**: 두산 로보틱스 (Doosan Robotics)  

### 프로그램 소개

ROS2 기반 로봇 제어 및 Linux 환경 개발 집중 교육
- Ubuntu 22.04 환경에서 C/C++ 개
- 실시간 센서/카메라 데이터 처리
- 멀티스레드, 네트워크 통신, SLAM 구현

## 🛠️ 사용 기술 및 도구

### 개발 환경
```
OS:           Ubuntu 22.04 LTS (전용 노트북)
ROS2:         Humble
C++ 표준:     C++17/20
Python:       3.9 이상
CMake:        3.16 이상
```

### 하드웨어
```
개발 노트북:  Intel i7, 16GB RAM, SSD
카메라:       Intel RealSense D435i (RGB-D)
로봇:         Doosan Collaborative Robot M0609
센서:         IMU, 거리센서, 압력센서
```

### 소프트웨어 스택
```
Core:         C++, Python, ROS2 Humble
Vision:       OpenCV, cv_bridge
Robotics:     MoveIt2, TF2, SLAM
Tools:        CMake, Colcon, Git, VSCode
Debug:        GDB, RViz, rqt_graph, rqt_plot
```
